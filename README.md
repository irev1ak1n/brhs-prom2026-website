# BRHS Prom 2026 Website

Student-built website created by a team to organize and promote BRHS Prom 2026, helping students access tickets, event details, and important updates in one place.

## Features
- Homepage hero section
- Live countdown to prom night
- Event location and theme details
- Ticket call-to-action
- Laravel + Blade project structure

## Project Purpose
This project was created to support prom planning and improve how students access information in one central place.
